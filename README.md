# unbound-mystery-gift-automator
A Python script for automating Pokemon Unbound mystery gift entry. Tested on Windows 11 with Python 3.13 with mGBA. Any emulator should work in theory; change focus_mgba() to the correct string and key mapping as needed.

# Requirements
```
pydirectinput=1.0.4
pygetwindow=0.0.9
```

# Execution 
1) Stand in front of mystery gift NPC
2) Run `python.exe mystery_gift.py code1 [code2 code3 code4 code5]`
